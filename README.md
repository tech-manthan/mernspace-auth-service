## Authentication Service

Authentication Service has following features :

1. Authentication & Authorization
   - User Registeration
   - User Login
   - Self
   - Refresh Tokens
   - Logout
   - Authenticate Middleware
   - CanAccess Middleware

2. Tenants
   - Create
   - GetAll
   - GetOne
   - Delete
   - Update

3. Users
   - Create
   - GetAll
   - GetOne
   - Delete
   - Update
